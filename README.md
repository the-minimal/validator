Minimal validation library in TypeScript.

# Install

```bash
yarn add @the-minimal/validator
```

# Features

- **Simple**
	- Validation only
- **Small**
	- 500 bytes bundle
	- Three-shakeable
- **Fast**
    - Parsing
    - Initialization
    - Runtime execution
    - Type checking
- **No stack traces by default**
- **No nested error messages by default**
- **No schema modifications**
- **No async**
- **No code generation**
- **No dependencies**
